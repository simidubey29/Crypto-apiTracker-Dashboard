# 🚀 Cryptocurrency Tracker Dashboard

A real-time Cryptocurrency Tracker built using Python, Streamlit, Pandas, Plotly, and CoinGecko API.

The dashboard fetches live cryptocurrency market data and provides interactive visualizations, market insights, and cryptocurrency comparisons.

---

## 📌 Problem Statement

Cryptocurrency prices change continuously throughout the day. Monitoring multiple cryptocurrencies across different websites can be time-consuming and inefficient.

This project provides a centralized dashboard that allows users to track cryptocurrency prices, market capitalization, trading volume, and top gainers in real time.

---

## 🎯 Project Objectives

- Fetch live cryptocurrency market data
- Display current prices and market information
- Identify top-performing cryptocurrencies
- Compare market capitalization and prices
- Visualize market trends
- Allow users to export data for analysis

---

## ✨ Features

### Live Cryptocurrency Data

Displays real-time cryptocurrency market information directly from the CoinGecko API.

### Search Cryptocurrency

Users can search any cryptocurrency by name.

### Top 5 Gainers

Automatically identifies cryptocurrencies with the highest 24-hour growth.

### Price Comparison Chart

Compares cryptocurrency prices using interactive bar charts.

### Market Capitalization Analysis

Visualizes market capitalization for top cryptocurrencies.

### CSV Download

Allows users to download cryptocurrency data for further analysis.

### Interactive Dashboard

Built with Streamlit and Plotly for a responsive user experience.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/crypto-tracker.git
```

Move into the project folder:

```bash
cd crypto-tracker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
